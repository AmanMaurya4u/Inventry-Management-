# 📦 Inventory Management System

A modern Inventory Management System built using **C# (.NET)** for the backend and **HTML, CSS, JavaScript** for the frontend.

The system helps businesses efficiently manage products, stock, suppliers, purchases, sales, and inventory reports.

## ⚙ Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/Inventory-Management.git
```

### Navigate to Project

```bash
cd Inventory-Management
```

### Restore Packages

```bash
dotnet restore
```

### Run the Application

```bash
dotnet run
```

---

## Database Setup

1. Open SQL Server.
2. Create a new database.
3. Update the connection string in `appsettings.json`.
4. Apply Entity Framework migrations (if using EF Core).

```bash
dotnet ef database update
```
